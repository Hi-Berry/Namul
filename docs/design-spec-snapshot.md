## PART A. 기획서 사양 (캡처본)

### 1. 시간
- 활동 시간 06:00~24:00 (18시간). 실제 0.5초 = 게임 1분 (하루 = 실시간 9분).
- 시간 정지: NPC 대화 / 상점 / 인벤토리 / 설정 중.
- 취침: 24시 이전 수동 → 다음날 06:00 / 24시 초과 강제 → 다음날 10:00. (취침 시 체력, 신력, 기력이 모두 최대치로 완전 회복)
- 계절: 30일/계절, 봄→여름→가을→겨울 순환. 약초 테이블에 영향.
- 요일: 평일(1~4) 탐험·농사·전투·호감도 / 장날(5,10) 오전 장터·오후 주막.

### 2. 스태미나 / 행동 비용
- 최대 100. 0~10 구간 → "탈진" 디버프, 이동속도 −50%.
| 행동 | 기력 | 시간 |
|---|---|---|
| 맵 이동 | 0 | 실시간 |
| 채집 일반(호미 3회) | 6 | 실시간 |
| 채집 희귀(6회) | 12 | 실시간 |
| 채집 명품(9회) | 18 | 실시간 |
| 전투 | 0 | −30분 |
| 주막 영업 | 시작 시 20 (저녁 45 / 밤 70) | 12시간(12~24시) |
| 메밀 수확 | 2 | 실시간 |

### 3. 캐릭터 스탯
- 일상: 기력(0~100, 11 미만 탈진).
- 전투: HP / MP(신력, 광역 마법 자원) / 민첩(은닉, 무기 무게로 턴 순서 결정).
- **장비 슬롯 3종**:
  - 무기: 호미/낫/절굿공이/부지깽이 — 공격력·기력효율·턴속도.
  - **의상(코스튬): 평민/비단 한복 — 기력소모 효율·이동속도.**
  - **장신구: 노리개/부적 — 최대 MP 증가·몬스터 드롭률.**

### 4. 맵 / 구역
- A. 내 집(안전): 취침·저장·인벤토리, 외부 메밀밭(향후 펫 관리 공간).
- B. 마을(안전): 무당·대장장이·촌장·식당·**의상점·건축상점**·주민, 장날 좌판(5/10일 오전).
- C. 산(위험): 던전형 별도 맵, 심볼 인카운터, 계절별 약초 리스폰, 입구 충돌 진입.
- 연결: 집↔마을, 마을↔산 입구(경계 트리거).

### 5. 농사
- 약초 3등급(호미 횟수): T1 3회/6기력, T2 6회/12, T3 9회/18. 계절별 스폰.
- 계절 약초 테이블 존재(구체 명·확률 TBD).
- 메밀(수동 작물): 장날 오전 종자 구매(가격 TBD) → 심기 → 4일 성장(무관리) → 수확 2기력(수확량 TBD) → 주막에서 소모.

### 6. 교역 / 시장
- 장날 5·10일. 오전(06~12) 종자/장비/고정상점. 오후(12~24) 주막 특별영업.
- 주막(타이쿤): 시작 시 기력 20 차감. 낮/저녁/밤 3페이즈 진행. 마감 시점에 따라 기력 차등 차감 (낮 20 / 저녁 45 / 밤 70). 하단의 '마감하기' 버튼을 통해 상시 조기 마감 및 정산 팝업 연동.
- 손님 웨이브: 고정 웨이브 순차, 웨이브 전원 완료 후 다음.
- 요리: 레시피에 표기된 순서대로 재료를 클릭하여 담고 [완성] 버튼을 눌러 서빙.
- 오류: 레시피 불일치 또는 재료 부족 -> 재료 소모 및 보수 0냥.
- 보너스: 빠른 서빙 배율 및 요리 숙련도 보너스 적용. 가격과 명성은 투입된 나물의 등급(Tier 1~3)에 따라 차등 보정 (T1: ×1.0, T2: ×1.3, T3: ×1.75).

- 심볼 인카운터(기력 충분 시 회피 가능). 턴 순서=민첩+무기 무게. 전투당 −30분.
